# VS Code Agents Instructions

This file provides instructions for AI coding agents working with the VS Code codebase.

## Atrisa App Rule

This checkout builds the standalone `Atrisa.app` distribution with the Atrisa
extension baked in as a built-in extension. Do not install Atrisa into the
user's normal VS Code profile and do not use regular VS Code as the Atrisa
runtime. Build, bake, and smoke-test `Atrisa.app` from this repo instead.

For detailed project overview, architecture, coding guidelines, and validation steps, see the [Copilot Instructions](.github/copilot-instructions.md).
